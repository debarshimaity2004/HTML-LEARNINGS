Day 1 — What is HTML?

On this day, we learned about essential HTML tags, including <h1>–<h6> for headings, <p> for paragraphs, <br> for line breaks, <ol> and <ul> for lists with <li> items, <a> for hyperlinks, and <img> for images.

Day 2 — HTML Nested Lists & Tables

On this day, we learned about lists and tables. For lists, we explored:

  • Ordered lists using <ol>
  • Unordered lists using <ul>
  • Nested lists combining <ol> and <ul>

For tables, we worked with:

  • <tr> — table row
  • <td> — table data cell
  • <th> — table header cell (used instead of <td> in the first row)

Day 3 — File Path in HTML

On this day, we learned about file pathing in HTML. There are mainly two types:

  • Relative Path — points to a file in relation to the current file’s location.
  • Absolute Path — provides the complete URL or full path to the file.

We also explored the HTML boilerplate, which is the basic structure every HTML document starts with. In addition, we learned about:

  • The <div> tag for grouping content.
  • Using class and id attributes to style or reference elements uniquely.

Finally, we understood that a typical website is mainly written in three parts:

  1. Header — the top section, usually for navigation or branding.
  2. Main — the central section where most of the content is displayed.
  3. Footer — the bottom section for extra information, links, or credits.

Day 4 — HTML forms

On this day, we learned about HTML forms. A form is created using the <form> tag, along with supporting tags like:

  • <label> — to describe input fields
  • <input> — to collect user data (text, number, email, etc.)
  • <select> and <option> — to create dropdown lists
  • <button> — to submit or reset the form

Rohit Bhaiya gave a very clear and practical explanation, even connecting how forms interact with databases to store and retrieve user information.

Day 5 — Deploy Your First Project

On this day, we learned about different media tags in HTML, such as:

  • <audio> — to embed audio files
  • <video> — to embed video files
  • <iframe> — to embed external content like other web pages or YouTube videos

We also practiced how to embed a YouTube video directly into our webpage using the <iframe> tag.

Finally, we learned how to deploy a project online using Netlify, where we can simply drag and drop our project folder to make it live on the internet.
