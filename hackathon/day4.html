HTML Forms

The fundamental truth is that a website isn't just a brochure for you to read. For the web to be useful, it needs a way to collect information from the user and send it back to the server. Without this, you couldn't log in, search for a video, buy a product, post a comment, or send a message. The web would be a read-only library.

Lets get started with the Tags and important porperties they carry!

The Form Tag (<form>)

The Purpose: The <form> tag is used to collect user input. It acts as a container for form elements like text fields, checkboxes, buttons, and more. Anything related to forms must be inside this Tag(like we used <table> for tables).

The Label Tag (<label>)

The Purpose: The <label> tag is used to define labels for form elements. Clicking a label focuses the associated input, making forms more accessible and user-friendly.

Common Attribute:
for: Links the label to an input element by matching its id.

Example: <label for="username">Username:</label>

The Input Tag (<input>)

The Purpose: The <input> tag is used to create interactive controls inside a form. It supports many type values such as text, password, email, checkbox, radio, and submit.

Common Attributes:
type: Defines the input type (text, password, radio, number, submit, checkbox, file, etc.).
name: Name of the input (used to send data to the Database/server).
value: The default value of the input.(used mostly in radio)
id: Unique identifier, often used with <label>.

Example:
<label for="email">Email:</label>
<input type="email" id="email" name="email"><br><br>

<label for="pwd">Password:</label>
<input type="password" id="pwd" name="pwd"><br><br>

<input type="submit" value="Login">





Placeholder: The attribute shows a short hint or example inside the input box before the user types anything.

Example: <input type="text" placeholder="Enter your name">

The Select, Option, and Button Tags

What they are:
- <select>: Creates a drop-down list.
- <option>: Defines each item inside the drop-down.
- <button>: Creates a clickable button.(mostly submit and reset)

Example (Dropdown):
<select name="fruits">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="cherry">Cherry</option>
</select>

Example (Button):
<button type="submit">Submit</button>

The Textarea Tag

What it is: <textarea> is used when you need multi-line text input, like comments or feedback forms.

Example:
<textarea name="message" rows="4" cols="30">
  Write your message here...
</textarea>
