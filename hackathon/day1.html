Day 1 — What is HTML?


Lets Gets Started with the Learnings from this above video!

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: This refers to the ability to create links that connect one web page to another, forming a connected “web” of information.
  • Markup Language: This means using “tags” to surround content, giving it structure and meaning. In other words, you are “marking up” plain text to turn it into a well-defined document.

The Headings

The Purpose: To structure your document in a way that is understandable to both humans and machines (like search engines and screen readers). It is notjust for making text big; it is for giving the text structural importance.

The Paragraph Tag(<p>)

The Purpose:To define a distinct paragraph of text. Browsers automatically add a little bit of space before and after a <p> element, separating it from other content. You should not use multiple line breaks; you should use multiple <p> tags.


The Horizontal Rule Tag (<hr>)

The Purpose: The <hr> tag is used to create a thematic break (a horizontal line) between sections of content. It is a self-closing tag (doesn’t need a closing tag).

Example: <hr> will display a horizontal line across the page.You can see it in the above portion of the page.


The Anchor Tag (<a>)

The Purpose: To create hyperlinks that connect one page to another, or to external websites. The href attribute specifies the link’s destination. Example: <a href="https://youtube.com">Visit Youtube</a>


The Line Break Tag (<br>)

The Purpose: To insert a single line break within text. Unlike paragraphs, <br> does not add spacing before or after — it just moves the content to the next line. Example: First Line<br>Second Line


The Image Tag (<img>)

The Purpose: To embed images into a webpage. The src attribute specifies the image source, and the alt attribute provides alternative text for accessibility. Example: <img src="photo.jpg" alt="sample image">

Common Anchor (<a>) Tag Attributes

href: Defines the destination URL of the link.
target="_blank": Opens the link in a new tab/window.
target="_self": Opens the link in the same tab (this is the default behavior).

Example: <a href="https://youtube.com" target="_self">Visit Youtube</a>
